# DNA-Vault
A Python-powered digital-to-biological data transcoder. This project demonstrates the future of cold-storage archiving by mapping binary data (00, 01, 10, 11) to DNA nucleotides (A, C, G, T). Featuring a Streamlit-based interface, Hyper-Vault allows users to encode digital files into genetic sequences and reconstruct them with bit-perfect accuracy.
HYPER-VAULT: DNA Data Archiving EngineDigital-to-Biological Cold StorageHYPER-VAULT is a high-prestige Python application that demonstrates the future of data storage by encoding digital information into synthetic DNA sequences. Using the four bases of life—Adenine (A), Cytosine (C), Guanine (G), and Thymine (T)—this tool allows for the transformation of binary data into a biological format and back with bit-perfect accuracy.🚀 How It WorksThe engine uses a 2-bit mapping system to translate binary data into nucleotides:00 $\rightarrow$ A (Adenine)01 $\rightarrow$ C (Cytosine)10 $\rightarrow$ G (Guanine)11 $\rightarrow$ T (Thymine)✨ FeaturesDigital-to-DNA Encoding: Upload any .txt file and instantly generate its genetic sequence.Biological-to-Digital Decoding: Paste a DNA string to reconstruct the original digital file.Celebratory UI: Integrated Streamlit interface with visual feedback and download capabilities.🛠️ Installation & UsageClone the repository:Install dependencies:Run the Vault:🧠 Why DNA Storage?Standard silicon-based storage lasts decades; DNA can last thousands of years if kept cool and dry. Furthermore, DNA is incredibly dense: you could theoretically store all the world's current data in a few grams of DNA.📄 LicenseThis project is licensed under the MIT License.
